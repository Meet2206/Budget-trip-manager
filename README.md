### Installation

1. **Clone the repository**:

   ```bash
   git clone https://github.com/Meet2206/budget-trip-manager.git
   cd budget-trip-manager
   ```

2. **Install dependencies**:

   Using npm:
   ```bash
   npm install
   ```

   Or using yarn:
   ```bash
   yarn install
   ```

3. **Start the development server**:

   Using npm:
   ```bash
   npm run dev
   ```

   Or using yarn:
   ```bash
   yarn dev
